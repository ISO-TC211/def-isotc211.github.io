Codelist vocabs hosted by OGC

May 15, 2025

The ISO/TC 211’s GOM has worked with the OGC to host many of the TC’s standards' codelists as Semantic Web vocabularies using the OGC’s reference data platform known as the 'OGC Rainbow'.

Figure 1. The OGC’s Rainbow

GOM, officially the Advisory Group 6, Group for Ontology Maintenance - has worked with the Open Geospatial Consortium (OGC) to take updated versions of standards' codelists that GOM has converted from their original XML into RDF data according to the SKOS vocabulary data model and load them into the OGC Rainbow which then automatically generates different views of them, including web pages.

The long-standing proposed persistent web addresses for each codelist can now be resolved - clicked on like any URL - and they will redirect to the Rainbow’s human-readable web page form of them.

For example, the widely used ISO19115-1’s Role Codes codelist is online at:

http://def.isotc211.org/19115/-1/2018/CitationAndResponsiblePartyInformation/code/CI_RoleCode

Visit the catalogue of all the codelists so far published as Semantic Web vocabularies here:

New PID IRI redirection service implemented

December 15, 2025

We have implemented a new server to for redirections https://def.isotc211.org redirections.

This allows redirections to elements such as codes within codelists, e.g. https://def.isotc211.org/common/RoleCode/editor to the editor code, not just to the whole Code Set.
